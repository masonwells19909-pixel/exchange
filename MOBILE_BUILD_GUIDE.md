# دليل تحويل الموقع إلى تطبيق أندرويد (APK)

هذا الدليل يشرح كيفية تحويل مشروع "تبادل" إلى تطبيق أندرويد حقيقي يمكنك تثبيته على هاتفك.

## المتطلبات
1. جهاز كمبيوتر (Windows, Mac, أو Linux).
2. برنامج **Android Studio** (مجاني من جوجل).
3. **Node.js** مثبت على جهازك.

## الخطوات

### 1. تحميل المشروع
قم بتحميل ملفات المشروع بالكامل على جهازك.

### 2. تثبيت المكتبات
افتح موجه الأوامر (Terminal) داخل مجلد المشروع واكتب:
```bash
npm install
```

### 3. بناء نسخة الويب
قم بإنشاء ملفات الموقع النهائية:
```bash
npm run build
```
سيظهر لك مجلد جديد اسمه `dist`.

### 4. إعداد Capacitor (محرك التطبيقات)
لقد قمتُ مسبقاً بإعداد Capacitor في المشروع. فقط قم بتنفيذ الأوامر التالية لإضافة منصة الأندرويد:

```bash
npx cap add android
npx cap sync
```

### 5. فتح المشروع في Android Studio
الآن، افتح المشروع داخل بيئة تطوير أندرويد:
```bash
npx cap open android
```
سيفتح لك برنامج Android Studio تلقائياً.

### 6. استخراج ملف APK
داخل Android Studio:
1. انتظر حتى ينتهي التحميل (Syncing).
2. من القائمة العلوية، اختر **Build** > **Build Bundle(s) / APK(s)** > **Build APK(s)**.
3. عند الانتهاء، ستظهر رسالة في الأسفل، اضغط على **locate** لتجد ملف الـ APK.

### 7. التثبيت
انقل ملف الـ APK إلى هاتفك وقم بتثبيته!
